Rapport Final - Test d'Intrusion bWAPP

Introduction

Cet apport final présente les résultats des tests d'intrusion effectués sur la plateforme bWAPP (http://10.161.11.142/bWAPP/portal.php). Ce document inclut une analyse des vulnérabilités découvertes, les méthodes d'exploitation, et des graphiques représentant les données recueillies.

Résumé des Vulnérabilités

Voici un résumé des principales vulnérabilités détectées durant les tests :

  • Local File Inclusion (LFI)
  • Cross-Site Scripting (XSS)
  • SQL Injection (SQLi)
